Revisão de Condicionais em Java

Este repositório contém exercícios de revisão sobre estruturas condicionais em Java, fundamentais para tomada de decisão em programas.

O objetivo é reforçar a compreensão de como executar trechos de código de forma condicional, aplicando if, else, else if e switch.

Conteúdo dos exercícios

If / Else – execução de blocos de código com base em condições booleanas

Else If – tratamento de múltiplas condições

Switch / Case – escolha entre várias opções de forma organizada

Operadores relacionais e lógicos – >, <, ==, !=, &&, ||

Exercícios práticos

Verificar se um número é positivo, negativo ou zero

Identificar se uma letra é vogal ou consoante

Determinar a classificação de idade (criança, adolescente, adulto, idoso)

Menu de opções usando switch
